## üì¶ Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Modern web browser with camera support

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/VineshThota/new-repo.git
   cd new-repo/qualitybrand-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

## üéÆ Usage Guide

### Getting Started

1. **Set Up Your Profile**
   - Enter your name and company information
   - This information will be included in generated reports

2. **Perform Quality Inspection**
   - **Upload Image**: Click "üìÅ Upload Image" to select a file
   - **Use Camera**: Click "üì∑ Use Camera" for real-time capture
   - Wait for AI analysis (2-3 seconds)

3. **Review Results**
   - Check the overall quality status (PASS/FAIL)
   - Review confidence level and detected defects
   - Examine dimensional analysis and recommendations

4. **Generate Professional Report**
   - Click "üìÑ Generate Professional Report"
